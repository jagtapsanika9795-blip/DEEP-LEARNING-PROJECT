- ## Project Objective

The objective of this project is to implement a deep learning model for image classification as part of the Data Science internship at Codtech IT Solutions. The project focuses on applying deep learning concepts to classify handwritten digit images and evaluate model performance using appropriate metrics and visualizations.

## Technologies Used

- Python 3.11.9  
- TensorFlow  
- NumPy  
- Matplotlib  
- Deep Learning (Neural Networks)

- ## Task Description

As part of Internship Task-2, a deep learning model was developed using TensorFlow to perform image classification. The MNIST dataset was used to train and test the model. The task involved data preprocessing, model building, training, evaluation, and visualization of results.
